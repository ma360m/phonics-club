# Product Images

Place product images in this folder using the **slug** as the filename.

## Naming convention

```
{product-slug}.jpg
```

Examples:
- `teachers-book-coloured.jpg`
- `jp-pupil-book-1-colour.jpg`
- `fun-phonics-pack-1.jpg`
- `jolly-dictionary.jpg`

## Supported formats

`.jpg`, `.jpeg`, `.png`, `.webp`

## Logo

Place your site logo at:

```
public/logo.png
```

(or `public/logo.svg`) — it appears in the header next to **PHONICS CLUB**.

Recommended size: **200×200px** minimum, transparent PNG preferred.

## Optional: category banners

- `category-jolly-phonics.jpg`
- `category-jolly-readers.jpg`
- `category-jolly-grammar.jpg`
- `category-teacher-resources.jpg`

After adding images, restart the dev server if needed.
